# Medium article library

81 ready-to-publish articles. Open `index.html` for the hub.

## Publish on Medium
1. Push this folder to GitHub and enable **GitHub Pages**.
2. Go to <https://medium.com/p/import>, paste an article's Pages URL, click **Import**.
3. Upload the matching video to YouTube and paste the link where the *Video:* placeholder is.

Each `*.html` has its diagram as a sibling `*.png` so the importer fetches it correctly.
